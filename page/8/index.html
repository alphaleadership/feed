2025-07-31
Pi-Hole

source

Pi-hole

Date de la fuite : 30 juillet 2025
Comptes affectés : 29 926
Site web : pi-hole.net

Description

In July 2025, a vulnerability in the GiveWP WordPress plugin exposed the names and email addresses of approximately 30k donors to the Pi-hole network-wide ad blocking project. Pi-hole subsequently self-submitted the list of impacted donors to HIBP.

Données exposées

Email addresses Names

Actions recommandées

  1. Changez votre mot de passe sur ce site et sur tout autre site où vous utilisez le même mot de passe
  2. Activez l’authentification à deux facteurs si disponible
  3. Surveillez vos comptes pour toute activité suspecte
