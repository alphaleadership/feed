# Adopt Me Trading Values - Fuite de données

**Date de la fuite** : 2022-07-01  
**Nombre de comptes affectés** : 86 136

## Description
In July 2022, the Adopt Me Trading Values website for assessing the value of pet trades within the "Adopt Me!" Roblox game suffered a data breach that was later redistributed as part of a larger corpus of data. The breach exposed 86k unique email addresses along with usernames and Roblox usernames, IP addresses and bcrypt password hashes. 

## Données exposées
- Email addresses
- IP addresses
- Passwords
- Usernames
