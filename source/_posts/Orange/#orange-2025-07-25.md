## Résumé de l'incident

Une fuite de données a été signalée concernant les clients d'Orange, exposant potentiellement des informations sensibles.

### Détails connus :
- **Date de découverte** : 25 juillet 2025
- **Type d'incident** : Fuite de données
- **Secteur** : Télécommunications

### Actions recommandées :
1. **Pour les clients Orange** :
   - Changez votre mot de passe Orange
   - Activez l'authentification à deux facteurs
   - Soyez vigilant face aux tentatives de phishing

2. **Surveillance** :
   - Vérifiez vos relevés bancaires pour toute activité suspecte
   - Surveillez vos emails pour des communications inhabituelles

3. **Pour plus d'informations** :
   - Contactez le service client Orange via les canaux officiels
   - Consultez le site officiel d'Orange pour les mises à jour
