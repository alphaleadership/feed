# Sumo Torrent - Fuite de données

**Date de la fuite** : 2014-06-21  
**Nombre de comptes affectés** : 285 191

## Description
In June 2014, the torrent site Sumo Torrent was hacked and 285k member records were exposed. The data included IP addresses, email addresses and passwords stored as weak MD5 hashes.

## Données exposées
- Email addresses
- IP addresses
- Passwords
- Usernames
- Website activity
